## Release Notes

This release introduces a major upgrade to the AI support experience across the platform. The new multi-turn chat interface allows users to have ongoing conversations with the AI, with context preserved across messages. Conversations can now be saved and reopened, and the UI includes improved loading states, error handling, copy actions, and a conversation history sidebar for easier navigation.

The AI orchestration layer has also been significantly improved. Questions are now routed more accurately to specialist agents for financial support, emotional wellbeing, cultural safety, regional services, and preterm care topics. Safety protections were strengthened with enhanced guardrails, stricter validation, output sanitization, structured logging, and improved streaming behavior in the agents API.

Additional work in this release includes new routing and offline test harnesses, architecture documentation, support for persistent conversation storage, and infrastructure updates required for the new chat and agent system to operate reliably.
